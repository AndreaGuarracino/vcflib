**vcfglxgt**

# SYNOPSIS

**vcfglxgt** [options] <vcf file>

# DESCRIPTION





# OPTIONS

```

options:
    -n, --fix-null-genotypes   only apply to null and partly-null genotypes

Set genotypes using the maximum genotype likelihood for each sample.

```
